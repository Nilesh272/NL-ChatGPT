# Observation Protocol (2 sessions)

**Task:** Complete a real work task using ChatGPT (or competitor) with screen share.  
**Duration:** 20–30 min per session  
**Do not coach** during the task; note behavior only.

---

## Setup

- Participant uses their normal tool/account
- Task examples: research a competitor, draft a client update, summarize a doc
- Observer notes timestamps

---

## Observe

| Signal | Notes |
|--------|-------|
| Skimming vs reading full answer | |
| Opens links / citations? | |
| Re-prompts when unsure? | |
| Copies without editing? | |
| Manual fact-check (Google, etc.)? | |
| Abandons tool mid-task? | |

---

## Post-task (5 min)

1. How confident are you in what you’ll send? (1–10)
2. What would have increased confidence?
3. Did you verify anything? How?

---

## Output

One-page summary per session → link in deck slide 4.
