## Key hypotheses to validate

1. Users trust fluent prose over cited evidence.
2. Users want a **checklist** more than a single trust score.
3. One bad AI mistake increases manual verification or tool abandonment.
4. High-stakes tasks need explicit human sign-off, not silent copy-paste.
